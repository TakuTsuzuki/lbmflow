# 要求定義書（ドラフト）：撹拌槽反応器 3D 連成マルチフィジックス LBM ソルバ

**文書ID**: REQ-M-F-STR / **位置づけ**: `docs/PLAN.md` の M-F（垂直機能）／ `ARCHITECTURE_V2.md`（次元×格子×精度×バックエンド×分割の直交設計）への上位要求
**対象コア**: `lbm-core2`（D3Q19/D3Q27, CpuScalar/CpuSimd/wgpu, MPI 分割）
**スコープ**: 回転インペラ駆動・下部ガススパージング付き 3D 撹拌槽における、速度場・ひずみ速度／偏差応力場・分散相負荷・多成分反応輸送の定量評価。単相→気液→反応を単一の連成タイムループで解く。

---

## 1. 対象問題の定義（物理コンフィギュレーション）

閉じた（または上部自由液面をもつ）3D 円筒／角柱容器。以下を同時に含む:

- **連続相**: 非圧縮・等温、Newton もしくは非 Newton（Carreau–Yasuda / Casson / power-law）液体。
- **分散気相**: 容器下部スパージャからの気泡群。密度比 `ρ_l/ρ_g ≈ 10³`、粘性比 `μ_l/μ_g ≈ 10²`。
- **回転インペラ**: 一定角速度 `Ω` の剛体回転翼（Rushton / pitched-blade 等の任意翼形）。容器壁は静止、必要に応じバッフル固定。
- **分散懸濁粒子**: 連続相にほぼ中立浮力で懸濁する微小固体粒子群。粒子が経験する履歴せん断負荷を評価対象とする。
- **反応スカラー**: 気相→液相への界面物質移動を伴う溶存成分、および液相内の一次／二次反応で消費・生成される複数の受動／能動スカラー。

対象レイノルズ数域 `Re = ρ N D²/μ ∈ [10³, 10⁵]`（遷移〜完全乱流）。目的の観測量は (a) 時間平均・位相平均された 3D 速度場、(b) ひずみ速度テンソル第二不変量に基づくせん断応力場 `γ̇(x)`、(c) 分散粒子のラグランジュ累積せん断曝露、(d) ガスホールドアップ `ε_g` と溶存スカラー濃度場。

---

## 2. 支配方程式系（連成対象）

```
連続相（低 Mach LBM で回収）:
  ∂ρ/∂t + ∇·(ρu) = 0
  ∂(ρu)/∂t + ∇·(ρuu) = -∇p + ∇·[ (μ + μ_t)(∇u + ∇uᵀ) ] + F_s + F_g + F_p + F_rot

二相界面（保存型 Allen–Cahn 相場）:
  ∂φ/∂t + ∇·(φu) = ∇·[ M ( ∇φ − (4/W) φ(1−φ) n̂ ) ],  n̂ = ∇φ/|∇φ|
  表面張力: F_s = μ_φ ∇φ   （μ_φ = 化学ポテンシャル, CSF 等価 σκn̂δ_s）

分散相粒子（Euler–Lagrange, 各マーカ）:
  m_p dv_p/dt = F_drag + F_buoy + F_lift + F_added-mass + F_pressure + F_wall

スカラー・反応輸送（成分 k, 移流拡散 + 源項）:
  ∂C_k/∂t + u·∇C_k = ∇·(D_k ∇C_k) + R_k(C) + Ṡ_k^{interface}

渦粘性（LES クロージャ）:
  μ_t = ρ (C_s Δ)² |S̄|,   |S̄| = √(2 S̄:S̄)
```

`F_g` 浮力（気相）、`F_p` 粒子反力（two-way coupling）、`F_rot` 回転系見かけ体積力（MRF 採用時のみ）。全項を単一の融合タイムループ（`step_band` 拡張）内で評価する。

## 3. 機能要求（数値手法）

### 3.1 基盤 LBM コア

- **FR-CORE-01**: 速度離散化は D3Q19 と D3Q27 を選択可能とする。高 We・強回転せん断域では等方性回復のため D3Q27 を既定とすること。
- **FR-CORE-02**: 衝突演算子に中心モーメント（cascaded）または cumulant LBM を実装し、既存 BGK/TRT/MRT と切替可能にすること。高 Re でのガリレイ不変性欠損・超粘性発散に対する数値安定性が受入条件（`τ → 0.5⁺` 近傍での破綻回避）。
- **FR-CORE-03**: 体積力の導入は Guo forcing を基本とし、相場・粒子・回転の各力源を重畳可能な単一 forcing インタフェースに集約すること。速度モーメントは `F/2` 補正込み。
- **FR-CORE-04**: 平衡分布は Hermite 多項式（Gauss–Hermite 求積）3 次までを保持し、圧縮性誤差 `O(Ma²)` を制御下に置くこと。運用 Mach 数 `Ma_lattice ≤ 0.1`。

### 3.2 乱流モデル（LES-LBM）

- **FR-LES-01**: SGS モデルとして Smagorinsky（動的 Germano 版を含む）と WALE を実装すること。壁近傍で `μ_t → 0` を正しく再現できる WALE を既定とする。
- **FR-LES-02**: 渦粘性を有効緩和時間 `τ_eff = τ_0 + 3 μ_t/(ρ c_s² Δt)` に反映し、SGS 応力を局所ひずみ速度テンソルから閉じること。`|S̄|` は非平衡モーメントから直接評価し、有限差分を用いないこと。
- **FR-LES-03**: 壁面せん断が支配的な翼先端・バッフル近傍では、`y⁺` に基づく壁関数または壁適合内挿境界を適用し、対数則層の未解像による壁せん断過小評価を回避すること。

### 3.3 回転インペラ（移動・回転境界）

- **FR-ROT-01**: 時間精度解法として direct-forcing 埋め込み境界法（IBM, Uhlmann 型）を実装すること。ラグランジュマーカ上で目標剛体速度 `U = Ω × r` を課し、正則化デルタ関数で forcing をユーラー格子へ散布・補間する。ガリレイ不変な移動境界を保証すること。
- **FR-ROT-02**: 定常近似の高速オプションとして MRF（回転参照系, frozen-rotor）を実装すること。回転ゾーンにコリオリ力 `−2Ω×u` と遠心力 `−Ω×(Ω×r)` を見かけ体積力として付加する。
- **FR-ROT-03**: 静止壁・バッフルの曲面・傾斜面には内挿バウンスバック（Bouzidi / Ginzburg–d'Humières の線形・二次補間）を適用し、階段近似誤差を除去すること。翼形は解析曲面または三角形メッシュ（STL）から距離場（signed distance）を生成して埋め込む。
- **FR-ROT-04**: インペラに作用するトルクと動力数 `Np = P/(ρ N³ D⁵)`（`P = 2πN·T`）を運動量交換法（MEM）または IBM forcing 積分から出力すること。吐出流量数 `N_Q` も算出する。
- **FR-ROT-05**（任意上位）: 翼–バッフル相対運動を厳密化する場合、sliding-mesh／overset（重合格子）の halo 補間を MPI 分割と両立させること。

### 3.4 高密度比二相流（ガススパージング）

- **FR-VOF-01**: 気液界面は保存型 Allen–Cahn 相場 LBM（Fakhari/Geier/Mitchell 系）で解くこと。既存 Shan–Chen 擬ポテンシャルは密度比・寄生流の制約から本用途では非採用とし、`ρ_l/ρ_g ≳ 10³` で質量保存誤差 `< 0.1%` を満たすこと。
- **FR-VOF-02**: 表面張力は well-balanced な化学ポテンシャル形式で導入し、静止液滴の Laplace 圧を `We→0` で寄生流速最小化（`Ca_spurious < 10⁻³`）で再現すること。無次元数 We, Eötvös(Eo), Morton(Mo), capillary(Ca) を物理条件へ整合させる。
- **FR-VOF-03**: スパージャ境界条件 — 容器底面の指定領域に、規定体積流量・気泡生成径の気相流入（相場 `φ=1` かつ速度流入）を課すこと。気泡の分裂・合体を相場の界面再構成で自然に許容する。
- **FR-VOF-04**（代替経路）: 気泡個々を解像しない大規模条件では、Euler–Lagrange 点気泡モデル（Tomiyama の抗力・揚力・付加質量・壁潤滑・乱流分散）を選択可能とすること。切替は `d_b/Δx` で自動判定してよい。
- **FR-VOF-05**: 界面物質移動 — 気液界面での溶存成分フラックスを Henry 則と Sherwood 数（`k_L a`）で閉じ、スカラー輸送へ源項 `Ṡ_k^{interface}` として渡すこと。

### 3.5 分散懸濁粒子

- **FR-PART-01**: 分散粒子は Euler–Lagrange 点粒子追跡を既定とし、Stokes 数 `St = τ_p/τ_f`、Schiller–Naumann 抗力則で運動方程式を積分すること。粒子体積分率が高い領域では two-way 結合を有効化する。
- **FR-PART-02**: 粒子径が格子オーダに達する場合、解像粒子法（partially saturated method / Noble–Torczynski, もしくは Ladd/Aidun-Lu）に切替可能とすること。
- **FR-PART-03**: 各ラグランジュ粒子の軌跡に沿って局所せん断速度 `γ̇` を時間積分し、累積せん断曝露 `∫γ̇ dt` と最大瞬間せん断 `max γ̇` を粒子属性として記録・出力すること。

### 3.6 スカラー・反応輸送

- **FR-SCAL-01**: 各成分の移流拡散を独立の ADE-LBM（D3Q7 または D3Q19 スカラー格子）で解き、速度場と単方向結合すること。Schmidt 数 `Sc = ν/D`、Péclet 数 `Pe` を無次元整合。
- **FR-SCAL-02**: 反応源項 `R_k(C)` を任意次数の反応速度式として与え、Damköhler 数 `Da` で制御すること。演算子分割（Strang splitting）で剛直な反応系を安定に積分する。
- **FR-SCAL-03**: スカラー境界条件に Dirichlet／Neumann／Robin（Danckwerts 型流入出）を実装すること。

### 3.7 応力場評価（本ソルバの中核出力）

- **FR-STRESS-01**: ひずみ速度テンソルを非平衡分布から局所評価すること:
  ```
  S_αβ = −1/(2 ρ c_s² τ_eff Δt) · Σ_i c_iα c_iβ f_i^{neq}
  ```
  有限差分を経由しない格子局所量として全セルで算出する。cumulant/MRT 採用時はせん断モーメントの緩和率で係数を補正する。
- **FR-STRESS-02**: 偏差応力テンソル `σ'_αβ = 2(μ+μ_t) S_αβ`、せん断速度 `γ̇ = √(2 S:S)`、第二不変量 `II_S`、von Mises 相当応力をフィールド出力すること。
- **FR-STRESS-03**: 壁せん断応力 `τ_w`（容器壁・翼面・バッフル面）を内挿境界の非平衡量から抽出し、面分布として出力すること。
- **FR-STRESS-04**: 非 Newton 連続相では `μ = μ(γ̇)`（Carreau–Yasuda 等）を局所せん断速度に応じて評価し、`τ_eff` へ反映する陰的整合を行うこと。

## 4. 離散化・無次元化・単位系要求

- **FR-COUP-01**: 全サブシステムを単一タイムステップ内で連成する。データフロー順序は「相場更新 → 密度/粘性場更新 → 力源合成 → 融合 collide-stream-moments → 境界 → スカラー ADE → 反応（split）→ 粒子積分」を既定とし、`probe_state_hash` 相当のビット等価テストで融合順序の妥当性を担保すること。
- **FR-COUP-02**: 無次元数マッチングを単位変換層で強制すること。物理条件から `Re, Fr, We, Eo, Mo, Sc, Da, St` を保存する格子パラメータを自動導出し、`Ma_lattice ≤ 0.1`・`τ_eff > 0.5`・拡散数/CFL 条件を満たさない場合は警告すること。
- **FR-COUP-03**: 非一様格子／局所細分化（block-structured refinement or AMR）を要求する。翼先端・スパージャ近傍・界面のせん断勾配を Kolmogorov 長 `η` に対し十分解像し、遠方バルクは粗格子とする。

## 5. 入出力・可視化要求

- **FR-IO-01**: 3D フィールド（`ρ, u, φ, γ̇, σ', τ_w, C_k, ε_g`）を VTK（構造格子, XML/VTU 圧縮）で時系列出力し、ParaView/VisIt で可視化可能にすること。
- **FR-IO-02**: 時間平均・位相平均統計 — インペラ回転位相でアンサンブル平均した平均場・RMS・レイノルズ応力を出力すること。
- **FR-IO-03**: Web GUI に 3D 表示（スライス面・等値面・せん断ヒートマップ）を追加すること。
- **FR-IO-04**: ラグランジュ粒子の累積せん断曝露ヒストグラム／CDF を集計出力すること。

---

## 6. 非機能要求

- **NFR-01（スケール）**: 完全乱流条件の解像には `O(10⁸–10⁹)` 格子点を要する。wgpu マルチ GPU ＋ MPI ドメイン分割で成立させること。
- **NFR-02（精度・メモリ）**: f32＋偏差格納（`f−w`）を既定とし、相場・スカラーの数値拡散を抑えつつ MLUPS を最大化すること。
- **NFR-03（性能）**: 融合カーネル（`step_band`）に相場・スカラー・forcing を統合し、リング二重化・SoA plane-major 配置の 3D 拡張を維持すること。
- **NFR-04（決定性）**: 並列縮約は決定的順序で畳み込み、リグレッションのビット等価比較を可能にすること。

## 7. 検証・受入基準（VALIDATION.md への追加項目）

- **VR-STR-01（単相撹拌）**: Rushton 翼で動力数 `Np` が実験相関（乱流域 `Np ≈ 5.0±0.5`）に一致。翼吐出速度プロファイルを PIV/LDA 基準データと `L∞rel` で照合。
- **VR-STR-02（ガススパージング）**: ガスホールドアップ `ε_g` と Sauter 平均径 `d_32` の妥当性。単一気泡終端上昇速度が Grace 線図（Eo–Mo–Re）に整合。
- **VR-STR-03（せん断場）**: Couette／円管ポアズイユの解析せん断解に対し局所ひずみ速度が `L∞rel ≤ 10⁻³`。壁せん断 `τ_w` を厳密解と照合。
- **VR-STR-04（スカラー/反応）**: Taylor–Aris 分散、既知 `Da` の反応拡散前線速度、`k_L a` を解析／相関解と照合。
- **VR-STR-05（連成回帰）**: 全力源 ON の縮小系で `probe_state_hash` ビット等価と保存則ドリフト上限を担保。

| # | 難所 | リスク | 緩和策 |
|---|---|---|---|
| 1 | 高密度比二相 (`10³`) | 相場界面の数値不安定・寄生流 | well-balanced 相場＋D3Q27、点気泡モデルの代替経路 |
| 2 | 高 Re 安定性 | BGK 発散、超粘性 | cumulant/中心モーメント＋WALE-LES |
| 3 | 回転境界の保存性 | IBM の運動量漏れ・トルク誤差 | multi-direct-forcing 反復、overset 検証 |
| 4 | 連成剛直性 | 反応・界面・回転の時間スケール乖離 | 演算子分割、サブサイクリング |
