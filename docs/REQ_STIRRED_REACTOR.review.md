# REQ_STIRRED_REACTOR.draft.md 敵対的レビュー

対象: `docs/REQ_STIRRED_REACTOR.draft.md`

## 指摘一覧

1. **重大度: Critical / 該当要求ID: 支配方程式, FR-STRESS-01, FR-LES-02 / 問題:** 非平衡モーメントからのひずみ速度評価式が BGK の標準形として誤っている。LBM では非平衡運動量流束は概ね `Π^neq = -2 ρ c_s^2 τ_lbm Δt S` ではなく、Chapman-Enskog 上の粘性係数 `ν = c_s^2(τ-1/2)Δt` と forcing 補正に依存するため、実装式は `τ_eff` か `τ_eff-1/2` か、post/pre-collision か、Guo forcing を含むかで係数が変わる。現行式のままではせん断率を系統的に過小/過大評価し、`τ→0.5` 近傍で破綻する。 / **推奨修正:** 使用する分布（pre-collision, regularized, central moment, cumulant）ごとに `Π^neq` と `S` の変換式を明記する。BGK なら、格子単位での粘性緩和時間、`(1-1/(2τ))` 型の離散時間補正、Guo forcing の半ステップ補正を含めた検証済み式に置き換える。

2. **重大度: Critical / 該当要求ID: FR-CORE-03, FR-STRESS-01 / 問題:** Guo forcing の `F/2` 補正を「速度モーメント」に入れる要求はあるが、非平衡応力テンソルからの `S` 評価時に forcing 由来の二次モーメント補正を差し引く要求がない。回転力、表面張力、浮力、粒子反力が強い本問題では、非平衡モーメントが粘性応力ではなく外力の離散化誤差を含む。 / **推奨修正:** `ρu = Σ_i c_i f_i + Δt F/2` だけでなく、`Π^neq` の定義に Guo forcing 項の二次モーメント補正を含める。応力出力は「純粘性応力」と「全運動量流束」を分離して定義する。

3. **重大度: Critical / 該当要求ID: FR-LES-02 / 問題:** `τ_eff = τ_0 + 3 μ_t/(ρ c_s² Δt)` は次元的に不整合である。動粘性 `ν = μ/ρ = c_s²(τ-1/2)Δt` なので、緩和時間への追加は `Δτ = ν_t/(c_s² Δt) = μ_t/(ρ c_s² Δt)` であり、D3Q19/D3Q27 の標準格子単位 `c_s²=1/3` を代入した場合にのみ `3ν_t/Δt` となる。現在の式は `c_s²` と係数 3 を二重計上する。 / **推奨修正:** 一般式を `τ_eff = 1/2 + (ν_0+ν_t)/(c_s² Δt)`、または `τ_eff = τ_0 + ν_t/(c_s² Δt)` と明記する。格子単位 `c_s²=1/3, Δt=1` の簡略形は別記する。

4. **重大度: Critical / 該当要求ID: 支配方程式, FR-LES-01, FR-LES-02 / 問題:** 支配方程式の SGS モデルは Smagorinsky 式 `μ_t=ρ(C_sΔ)^2|S|` だけを示しているが、FR-LES-01 は WALE を既定としている。WALE の渦粘性は速度勾配テンソルの二乗テンソルの deviatoric symmetric part に依存し、非平衡モーメントの `S` だけからは閉じない。 / **推奨修正:** Smagorinsky と WALE の式を分離する。WALE では局所速度勾配を何で評価するかを明記し、「有限差分を用いない」との要求を撤回するか、LBM モーメントから速度勾配全体を復元する根拠を追加する。

5. **重大度: Critical / 該当要求ID: FR-ROT-02 / 問題:** MRF の見かけ力の符号と速度の定義が曖昧で、実装者が慣性系速度 `u` に `-2Ω×u` を掛ける危険がある。回転座標系の運動方程式に入るコリオリ項は回転系相対速度に対する `-2ρ Ω×u_rel`、遠心項は通常 `-ρ Ω×(Ω×r)` を左辺移項の体積力として扱うが、圧力に吸収するかどうかでも実装形が変わる。 / **推奨修正:** `u_abs`, `u_rel`, `r` の基準原点、回転ゾーン内外のインターフェース条件、密度係数を明記する。MRF では相対速度で解くこと、静止槽壁・バッフルには MRF を適用しないことを要求に入れる。

6. **重大度: Critical / 該当要求ID: FR-ROT-02, FR-ROT-01 / 問題:** MRF と IBM 回転翼の役割が競合している。MRF は「翼を静止させた回転座標系」の近似であり、同じ回転ゾーン内で direct-forcing IBM の移動翼 `U=Ω×r` を同時に課すと回転運動を二重計上する。 / **推奨修正:** 実行モードを `IBM in inertial frame`, `MRF frozen rotor`, `sliding/overset` に排他的に分ける。各モードで使う境界条件、出力トルク、平均化の扱いを個別定義する。

7. **重大度: Critical / 該当要求ID: 支配方程式, FR-VOF-02 / 問題:** 表面張力を `F_s = μ_φ ∇φ` と書いているが、化学ポテンシャル形式と CSF 形式の符号・スケーリング・圧力テンソルの定義が未整合である。一般には `F_s = -φ∇μ_φ`、`μ_φ∇φ`、または divergence of capillary stress がモデル依存で、符号を誤ると界面は収縮ではなく発散する。 / **推奨修正:** 採用する free-energy/phase-field モデルを一つに固定し、自由エネルギー、化学ポテンシャル、界面厚さ `W`、表面張力 `σ` との係数関係、力の符号を完全に記述する。CSF 等価性は検証項に落とす。

8. **重大度: Critical / 該当要求ID: 支配方程式, FR-VOF-01 / 問題:** 保存型 Allen-Cahn 方程式の形が不完全で、`n̂=∇φ/|∇φ|` のゼロ勾配正則化、`φ` の範囲、mobility `M` の次元、界面厚さとの関係がない。現行の `∇φ − (4/W)φ(1−φ)n̂` は符号規約や `φ∈[0,1]` 前提に敏感で、実装差で界面が逆拡散または過拡散する。 / **推奨修正:** Allen-Cahn の保存形を文献式どおりに固定し、`φ` の相定義、`n̂` の正則化 `|∇φ|_ε`、`M` と Peclet/Cahn 数、境界での接触角条件を要求する。

9. **重大度: Critical / 該当要求ID: FR-VOF-01 / 問題:** `ρ_l/ρ_g ≳ 10³` で相場 LBM が質量保存誤差 `<0.1%` を満たすという要求が強すぎるうえ、対象形状・時間・解像度・流量に依存する。保存型 Allen-Cahn は局所再初期化に近い項を持ち、トポロジー変化・開境界・流入流出では総気相体積の厳密保存を保証しない。 / **推奨修正:** ベンチマーク別に「閉じた静止液滴」「上昇気泡」「スパージャ開境界」で保存量を分け、時間、格子解像度、流出入量を含む許容誤差を定義する。

10. **重大度: Critical / 該当要求ID: FR-VOF-03 / 問題:** 下部スパージャで `φ=1` かつ速度流入を課すだけでは、規定体積流量と気泡生成径を同時に満たせない。相場法では気泡径は流入パターン、表面張力、接触角、ピンチオフ解像度で決まり、単純な Dirichlet 条件では圧力反射・質量不整合・非物理的合体を起こす。 / **推奨修正:** スパージャを「気相体積流量境界」「周期的/確率的気泡注入」「解像オリフィス」のどれで表すかを分ける。流量保存、圧力境界、接触角、最小 `d_b/W` と `d_b/Δx` を明記する。

11. **重大度: Major / 該当要求ID: FR-VOF-03 / 問題:** 「気泡の分裂・合体を相場の界面再構成で自然に許容」としているが、保存型 Allen-Cahn の界面厚さ・mobility・格子解像度に強く依存し、実気泡の薄膜排液や合体阻害を物理的には解かない。 / **推奨修正:** 分裂・合体は「数値的に許容」までに弱め、検証対象の気泡径分布を解像相場で扱うか、人口収支/点気泡モデルで扱うかを分ける。

12. **重大度: Critical / 該当要求ID: FR-VOF-04 / 問題:** 解像相場モデルと Euler-Lagrange 点気泡モデルの自動切替を `d_b/Δx` だけで決めるのは物理的に不十分。Eo, Mo, Re_b, Weber, ボイド率、界面物質移動、two-way coupling、BIT の有無でモデル適用範囲が変わる。 / **推奨修正:** 切替条件に `d_b/W`, `Eo`, `Re_b`, `α_g`, `We_b`, 物質移動モデルの一貫性を追加し、同一計算内の混在時は相間質量・運動量・スカラー保存則を定義する。

13. **重大度: Critical / 該当要求ID: FR-SCAL-01, FR-VOF-05, 支配方程式 / 問題:** スカラーを「速度場と単方向結合」とする一方、対象には能動スカラー、界面物質移動、反応による密度/粘性/表面張力変化の可能性が含まれる。単方向結合では Marangoni 効果、発熱・密度変化、反応で相物性が変わる系を表現できない。 / **推奨修正:** スカラーを受動・能動に分類し、能動スカラーではどの物性・力にフィードバックするかを明記する。受動のみなら「能動」を削除する。

14. **重大度: Major / 該当要求ID: FR-SCAL-01, FR-LES-01 / 問題:** ADE-LBM の分子拡散 `D_k` だけが要求され、LES の SGS スカラー拡散がない。乱流撹拌槽の `Sc` が大きい液相では、乱流 Schmidt 数 `Sc_t` による `D_eff = D + ν_t/Sc_t` がないと濃度場と `k_La` が過小混合になる。 / **推奨修正:** SGS スカラー流束モデルを追加し、`Sc_t` または動的 SGS スカラー拡散の既定値、壁・界面近傍での扱い、ADE-LBM の緩和時間への反映式を定義する。

15. **重大度: Major / 該当要求ID: FR-SCAL-02 / 問題:** 「任意次数の反応速度式」と「Strang splitting で剛直反応系を安定」は過大要求である。Strang splitting は反応 ODE 自体の安定性を保証しないし、負濃度・質量保存・化学量論制約の要求もない。 / **推奨修正:** 反応ソルバを陽解法、陰解法、Rosenbrock/BDF 等に分け、剛直性判定、負濃度クリッピング禁止/制限、元素保存誤差、split 誤差の受入基準を定める。

16. **重大度: Critical / 該当要求ID: FR-PART-01 / 問題:** 粒子 two-way coupling を「体積分率が高い領域」で有効化するとあるが、閾値も運動量反力の散布法もない。点粒子の Schiller-Naumann 抗力は粒子 Reynolds 数の適用範囲があり、中立浮力微粒子の履歴せん断評価では Saffman 揚力、Basset 履歴、Faxen 補正の要否が粒子径/格子幅に依存する。 / **推奨修正:** `α_p` または mass loading による one-way/two-way/four-way 切替、抗力則の `Re_p` 範囲、反力散布カーネル、運動量保存検証を要求する。

17. **重大度: Major / 該当要求ID: FR-PART-01, FR-PART-03 / 問題:** SGS 乱流分散が欠落している。LES 速度場でラグランジュ粒子を追跡すると、未解像乱流による粒子拡散とせん断曝露の分散が欠落し、`∫γ̇dt` の PDF/CDF が格子依存になる。 / **推奨修正:** stochastic SGS dispersion、resolved-only 明記、または DNS 解像時のみ粒子曝露を定量指標にする条件を追加する。

18. **重大度: Critical / 該当要求ID: FR-STRESS-02, FR-STRESS-04, FR-LES-02 / 問題:** 非 Newton 粘性 `μ(γ̇)` と LES 渦粘性 `μ_t` を同じ `τ_eff` に足す設計は二重計上・循環依存のリスクがある。`γ̇` は `τ_eff` から求め、`τ_eff` は `μ(γ̇)+μ_t` から決まるため、陰的反復・下限/上限なしでは発散または履歴依存になる。 / **推奨修正:** `μ_molecular(γ̇)` と `μ_t` の合成則、反復手順、収束基準、緩和、`τ_min/τ_max`、LES が非 Newton 乱流に適用可能な範囲を明記する。

19. **重大度: Major / 該当要求ID: FR-STRESS-02 / 問題:** `σ'_αβ = 2(μ+μ_t)S_αβ` を「偏差応力」としているが、非圧縮 Newton 流体の偏差粘性応力ならよい一方、多相・非 Newton・LES では相平均応力、SGS 応力、界面応力、粒子応力が混在する。von Mises 相当応力も流体せん断場の評価量として定義が曖昧。 / **推奨修正:** 出力を `resolved viscous stress`, `SGS stress`, `capillary stress`, `particle stress` に分ける。von Mises はどの応力テンソルから計算するかを限定する。

20. **重大度: Major / 該当要求ID: FR-STRESS-03 / 問題:** 内挿境界の非平衡量から壁せん断を抽出する要求は一般には成立しない。Bouzidi/Ginzburg 境界や IBM 近傍では分布関数が境界処理で再構成され、非平衡モーメントが物理的壁勾配を表さない場合がある。 / **推奨修正:** 壁せん断は壁面接線速度勾配の再構成、IBM forcing 積分、MEM などモード別に定義する。検証は Couette/Poiseuille だけでなく曲面移動壁で行う。

21. **重大度: Critical / 該当要求ID: FR-ROT-01 / 問題:** Uhlmann 型 direct-forcing IBM に「ガリレイ不変な移動境界を保証」と書いているが、古典的 IBM はデルタ補間、マーカ密度、質量行列、反復 forcing によりすべり・運動量漏れ・ガリレイ不変性誤差を持つ。保証要求としては虚偽に近い。 / **推奨修正:** 「保証」を削除し、移動壁 Couette、回転円柱、Taylor-Couette でのすべり速度、トルク誤差、運動量保存誤差の閾値を設定する。multi-direct-forcing または implicit IBM の採用条件を追加する。

22. **重大度: Major / 該当要求ID: FR-ROT-03 / 問題:** 内挿バウンスバックは静止壁向けに書かれているが、翼形は IBM で扱うのか距離場バウンスバックで扱うのかが競合している。 moving curved boundary に単純な Bouzidi 境界を適用すると Galilean invariance とトルク精度に問題が出る。 / **推奨修正:** 静止壁・バッフルは内挿 BB、移動翼は IBM または moving-wall interpolated BB と明確に分離する。STL 距離場の更新頻度と回転時の幾何誤差も定義する。

23. **重大度: Major / 該当要求ID: FR-ROT-04 / 問題:** 動力数 `Np = P/(ρN³D⁵)` の `ρ` が多相・非 Newton・ガスホールドアップ時に曖昧である。`P=2πN·T` は `N` が rev/s ならよいが、`Ω` rad/s と混同すると `2π` を二重計上する。 / **推奨修正:** `N=Ω/(2π)`、`D` はインペラ径、`ρ` は液相基準または混相平均のどちらかを明記する。ガス通気時は `Np_g` と非通気 `Np_0`、通気動力低下比を別出力にする。

24. **重大度: Major / 該当要求ID: FR-ROT-04 / 問題:** 吐出流量数 `N_Q` の定義がない。一般には `N_Q = Q/(ND³)` だが、Rushton では吐出面の選択、軸流/径流成分、正味流量か絶対流量かで値が変わる。 / **推奨修正:** `Q` の積分面、速度成分、時間平均/位相平均、逆流の扱いを定義する。

25. **重大度: Major / 該当要求ID: FR-COUP-02 / 問題:** 無次元数マッチングで `Re, Fr, We, Eo, Mo, Sc, Da, St` を「保存」するとしているが、LBM の低 Mach 制約、界面厚さ、重力、粘性比、密度比、拡散係数を同時に一致させる自由度は一般に足りない。音響スケーリングと非圧縮性の整合も未定義。 / **推奨修正:** 物理単位から格子単位への変換自由度を列挙し、保存する無次元数の優先順位と妥協ルールを定義する。`Ma`, `Ca`, `Cahn`, `Pe_phi`, `τ` 範囲を含めた feasibility check を必須にする。

26. **重大度: Major / 該当要求ID: 無次元数全般, FR-COUP-02 / 問題:** 無次元数の定義が不足している。`Fr` は撹拌では `N²D/g` または `Ω²D/g`、`We=ρN²D³/σ`、`Eo=Δρ g d_b²/σ`、`Mo=g μ_l⁴ Δρ/(ρ_l² σ³)`、`Ca=μU/σ`、`Pe=UL/D_m`、`Da` は反応次数ごとに異なる。現行では実装者が互換性のない定義を使う。 / **推奨修正:** 対象別の代表長さ・速度を表にし、各無次元数の式、使用物性、適用モデル、検証に使う値を明記する。

27. **重大度: Major / 該当要求ID: FR-COUP-02, FR-LES-02 / 問題:** `τ_eff > 0.5` を警告条件としているが、LES 渦粘性は `τ_eff` を増やす方向であり、問題は下限だけでなく上限にもある。大きすぎる `τ_eff` は過拡散・境界精度劣化・スカラーとの時間スケール不整合を生む。 / **推奨修正:** `τ_eff` の推奨範囲、クリッピング方針、クリッピング時の `ν_t` 診断、局所 Reynolds/CFL 制約を追加する。

28. **重大度: Critical / 該当要求ID: FR-COUP-01 / 問題:** 単一タイムステップ内の固定順序で相場、流体、スカラー、反応、粒子を更新する要求は、強連成・剛直反応・表面張力波・粒子衝突で安定性を保証しない。`probe_state_hash` は実装順序の回帰には有効だが、物理的妥当性の担保にはならない。 / **推奨修正:** 演算子分割誤差、サブサイクリング、反復強連成、capillary time step、粒子時間刻み、反応 ODE 時間刻みを要求に入れる。ビット等価は物理検証とは別扱いにする。

29. **重大度: Major / 該当要求ID: FR-COUP-03 / 問題:** AMR/局所細分化を必須としているが、標準 LBM は非一様格子・AMR で分布関数補間、時間刻みサブサイクル、保存性、境界 halo が難しい。`step_band` 融合カーネル、MPI、wgpu と同時に必須化すると実装計画が破綻する。 / **推奨修正:** 初期版では一様格子を基準にし、AMR は上位オプションへ降格する。AMR を必須にするなら coarse-fine interface の保存補間、時間ステップ比、検証問題を定義する。

30. **重大度: Major / 該当要求ID: FR-CORE-01, FR-CORE-04 / 問題:** D3Q27 を高 We・強回転で既定としつつ、D3Q19/D3Q27 双方で Hermite 3 次平衡を保持するとしている。D3Q19 は 3 次等方性や高次 Hermite 求積の表現に制限があり、D3Q27 と同等の三次平衡を前提にできない。 / **推奨修正:** 各格子で対応する平衡分布の次数と回収精度を分けて定義する。D3Q27 既定の条件を「多相・強 forcing・cumulant」など具体条件にする。

31. **重大度: Major / 該当要求ID: FR-CORE-02 / 問題:** cumulant/中心モーメントなら `τ→0.5⁺` で破綻回避できるという安定性主張が過大である。高 Re、強 forcing、多相密度比、低粘性では positivity、aliasing、界面力、境界条件が支配的に不安定化する。 / **推奨修正:** 安定性を保証ではなく対象ベンチマークでの許容範囲に落とす。緩和率、regularization、filtering、entropic limiter の有無を要求する。

32. **重大度: Major / 該当要求ID: NFR-02 / 問題:** f32 と偏差格納を既定にしているが、`ρ_l/ρ_g≈10³`、表面張力、圧力差、トルク積分、長時間保存量では丸め誤差が支配的になる可能性がある。寄生流 `Ca_spurious<10^-3` や質量保存 `<0.1%` と整合しない。 / **推奨修正:** f64 または mixed precision の要求を追加し、保存量・トルク・界面曲率・縮約は精度選択可能にする。f32 既定の適用範囲を単相/弱連成に限定する。

33. **重大度: Critical / 該当要求ID: 対象問題, FR-VOF-03, FR-IO-01 / 問題:** 上部自由液面と脱気が欠落している。問題定義は「閉じたまたは自由液面」とするが、スパージングでは気泡の上部排出、ヘッドスペース圧、液面変形、開境界がガスホールドアップと `k_La` を支配する。閉じた槽で気相流入だけを課すと気体が蓄積して非物理になる。 / **推奨修正:** 上部境界を閉鎖、自由液面、脱気 outlet のどれにするか必須指定にする。気相体積保存、流出フラックス、圧力境界、液面接触角を要求する。

34. **重大度: Critical / 該当要求ID: 支配方程式, FR-COUP-02 / 問題:** 静水圧・重力場の扱いが不足している。`F_g` を「浮力（気相）」とだけ書くが、二相密度場では全相に `ρg` を入れ、圧力勾配と静水圧を well-balanced に扱わないと、静止成層でも流れが発生する。 / **推奨修正:** 重力加速度、圧力分解（動圧/静水圧）、浮力近似か full density forcing か、well-balanced hydrostatic test を定義する。

35. **重大度: Major / 該当要求ID: FR-VOF-05, FR-SCAL-03 / 問題:** スカラーの壁・界面境界条件が不足している。Henry 則と Sherwood 数を源項にするとあるが、解像界面では界面ジャンプ条件、未解像点気泡では `k_La(C^*-C)`、壁では吸着/無流束/反応壁が必要で、同じ `Ṡ_interface` では閉じない。 / **推奨修正:** 解像界面と点気泡で物質移動モデルを分離する。界面での分配係数、法線フラックス、スカラーの相別拡散係数、壁境界を定義する。

36. **重大度: Major / 該当要求ID: FR-IO-01 / 問題:** `ε_g` を 3D フィールドとして出力するとあるが、相場の `φ` とボイド率 `ε_g` の関係が未定義。解像界面では局所 `φ` は体積分率ではなく拡散界面指標であり、点気泡モデルではカーネル平均が必要。 / **推奨修正:** `ε_g` の定義を解像相場・点気泡・混在モデルごとに分け、体積平均窓とフィルタ幅を明記する。

37. **重大度: Major / 該当要求ID: FR-IO-02, FR-ROT-02 / 問題:** 位相平均統計は MRF frozen-rotor と整合しない。MRF は相対位置を固定した定常近似で、翼-バッフル相対位相の時系列が存在しない。 / **推奨修正:** 位相平均は IBM/overset の非定常回転モードのみ必須にする。MRF では回転座標平均または疑似定常場として別出力にする。

38. **重大度: Major / 該当要求ID: VR-STR-01 / 問題:** Rushton `Np≈5.0±0.5` だけでは受入基準として粗い。槽径/翼径比、バッフル数、クリアランス、Re、自由液面/蓋、通気なし/ありで `Np` は変わる。PIV/LDA の `L∞rel` も閾値がない。 / **推奨修正:** 標準形状（例: baffled tank, `D/T`, `C/T`, blade geometry）、Re 範囲、非通気条件、許容誤差、速度プロファイルの測線と `L2/L∞` 閾値を定義する。

39. **重大度: Major / 該当要求ID: VR-STR-02 / 問題:** ガスホールドアップと `d_32` の「妥当性」に定量閾値がない。単一気泡 Grace 線図と撹拌槽多気泡スパージングは別問題で、合体・分裂・BIT・界面活性の有無で `d_32` が大きく変わる。 / **推奨修正:** 単一気泡、気泡群、撹拌槽通気を別検証に分ける。`U_t`, `ε_g`, `d_32`, `k_La` の許容相対誤差と実験相関を指定する。

40. **重大度: Major / 該当要求ID: VR-STR-03 / 問題:** Couette/Poiseuille で `L∞rel≤10^-3` は、曲面境界・IBM・LES・非 Newton・多相応力の検証には不十分かつ場合により過度に厳しい。`L∞rel` は壁近傍の格子配置で発散的に厳しくなる。 / **推奨修正:** 単相 BGK/MRT の製造解、曲面 Couette、回転円柱、非 Newton Poiseuille、多相静止液滴を分け、格子収束次数と `L2/L∞` を設定する。

41. **重大度: Major / 該当要求ID: VR-STR-04 / 問題:** Taylor-Aris、反応拡散前線、`k_La` の照合に閾値がない。`k_La` は解像界面からの面積積分か相関式かで意味が異なる。 / **推奨修正:** 各検証の許容誤差、対象 Pe/Da/Sc、境界条件、`k_La` の算出式を明記する。

42. **重大度: Major / 該当要求ID: VR-STR-05 / 問題:** `probe_state_hash` ビット等価は GPU、MPI、f32、非決定的縮約、AMR と相性が悪く、物理受入基準ではない。保存則ドリフト上限も数値がない。 / **推奨修正:** ビット等価は単一バックエンドの回帰に限定し、GPU/MPI では許容誤差ベースにする。質量、運動量、スカラー、気相体積、粒子数、エネルギー様量のドリフト閾値を個別に設定する。

43. **重大度: Minor / 該当要求ID: FR-IO-01 / 問題:** VTK の「構造格子, XML/VTU」とあるが、VTU は非構造格子であり、構造格子なら VTI/VTS が自然である。 / **推奨修正:** 一様格子は VTI、構造曲線格子は VTS、非構造/AMR は VTU/VTK AMR と明記する。

44. **重大度: Major / 該当要求ID: NFR-01, NFR-03 / 問題:** `O(10^8-10^9)` 格子に相場、複数スカラー、cumulant、粒子、IBM、AMR をすべて融合する要求はメモリ帯域・GPU メモリの見積もりがない。D3Q27 で複数分布を持つと 1e9 格子は現実的に数 TB 級になる。 / **推奨修正:** 格子あたりバイト数、分布数、スカラー数、粒子数、GPU メモリ、I/O 量、チェックポイント頻度の予算を要求に追加する。

45. **重大度: Major / 該当要求ID: 全体 / 問題:** 初期化・助走条件が欠落している。撹拌槽は初期静止からの spin-up、ガス導入開始、反応開始、統計平均開始時刻で結果が大きく変わる。 / **推奨修正:** 初期速度/圧力/相場/スカラー/粒子配置、インペラ ramp-up、ガス流量 ramp、統計サンプリング開始条件、準定常判定を要求する。

46. **重大度: Major / 該当要求ID: 全体 / 問題:** 気泡誘起乱流（BIT）が欠落している。点気泡モデルや未解像通気条件では、気泡 wake による乱流生成を LES の resolved/SGS だけでは表せない。 / **推奨修正:** 解像気泡では不要条件を明記し、点気泡では BIT 生成項または乱流分散・有効粘性補正モデルの採否を定義する。

47. **重大度: Major / 該当要求ID: 全体 / 問題:** 境界での濡れ性・接触角が未定義である。気液相場、スパージャ、槽壁、翼面、バッフルで接触角がないと、液面・気泡付着・壁面フィルムが格子依存になる。 / **推奨修正:** 壁ごとに接触角境界条件、滑り/非滑り、相場フラックス条件を要求する。

48. **重大度: Minor / 該当要求ID: 文書全体 / 問題:** 「D3Q19/D3Q27 選択可能」「D3Q27 既定」「D3Q7 または D3Q19 スカラー」「AMR 必須」など既定値が複数あり、最小実装構成が読めない。 / **推奨修正:** MVP、推奨、上位オプションを表で分ける。各検証がどの構成を対象にするかを明記する。

## 総合判定

この要求定義書をそのまま実装に渡すべきではない。現状は、物理モデル・LBM 離散化・乱流閉鎖・多相界面・回転座標系・検証基準が同じ粒度で混在し、いくつかの式は係数または符号の誤実装を誘発する。特に、非平衡応力評価、`τ_eff` と渦粘性の関係、MRF 見かけ力、表面張力の化学ポテンシャル形式、保存型 Allen-Cahn の仕様、スカラー/粒子/気泡の SGS モデル、上部脱気境界は Critical な再定義が必要である。

実装前に、少なくとも以下を改訂条件にすべきである。

- 実行モードを単相撹拌、解像気液、点気泡、反応輸送、粒子追跡に段階分けする。
- 各モードで使用する支配方程式、LBM 格子、衝突演算子、forcing、境界条件、単位変換を閉じた形で定義する。
- 無次元数と代表量を表で固定し、同時に一致できない場合の優先順位を定める。
- VR-STR-01〜05 を、形状・物性・格子解像度・時間・誤差ノルム・閾値付きの検証仕様に書き換える。
- 「保証」「自然に許容」「安定に積分」など検証不能な表現を、測定可能な誤差・保存則・適用範囲へ置き換える。
